# AcadAssist

## Overview

This is a frontend-only web application built using **Next.js**. The platform is designed to provide easy access to study materials for students across **all departments and all semesters**.

The use of Next.js ensures **fast performance**, **smooth navigation without page reloads**, and **efficient routing**.

## Features

- Access study materials for multiple departments  
- Organized content for all semesters  
- Bookmark feature to save important materials for quick access  
- Dark mode and light mode support  
- Contributor page to showcase past contributors  
- Fast and responsive UI  
- Seamless navigation (no full page reloads)  
- Clean and user-friendly interface

## Getting Started

### Prerequisites

Make sure you have installed:

- Node.js  
- npm or yarn  

### Installation

```bash
npm install
```

### Run the project

```bash
npm run dev
```

The app will run on:

```
http://localhost:3000
```
